## &lt;Point&gt;

The point series is used to render `<circle>` elements representing the values in the chart. Is often used in combination with the `Line` or `Area` series to highlight the data points on the chart.

### barConfig

For the bar series there is a global `barConfig` object on the `<Chart>` component that lets you modify the way multiple bar series are rendere together. It's used to define values like the space between bars or their width.  Have a look at the `BarConfig` type in the <Link href="/docs/components/chart/#api-barconfig">Chart API reference</Link>.
