# Curves

solid-charts uses <Link href="https://d3js.org/d3-shape/curve" newTab>D3's curves</Link> to generate the line path for the Area and Line series. You can use any of the available curve functions or provide your custom function to customize the appearance of your line chart. The default curve is `curveLinear`, which creates a straight path between points. The curve functions are re-exported from D3 under `solid-charts/curves`.

## Available curves

All the curves are documented in <Link href="https://d3js.org/d3-shape/curve" newTab>D3's curves documentation</Link>.

<Code code={`
  import {
    curveBasis,
    curveBasisClosed,
    curveBasisOpen,
    curveBumpX,
    curveBumpY,
    curveCardinal,
    curveCardinalClosed,
    curveCardinalOpen,
    curveCatmullRom,
    curveCatmullRomClosed,
    curveCatmullRomOpen,
    curveLinear,
    curveLinearClosed,
    curveMonotoneX,
    curveMonotoneY,
    curveNatural,
    curveStep,
    curveStepAfter,
    curveStepBefore,
  } from 'solid-charts/curves'
`} lang="tsx" />
